# pwn-env

## Install

Clone the repository

```sh
git clone https://github.com/d0ublew/pwn-env
cd pwn-env
```

Choose either one:

- pull from registry

```sh
docker pull d0ublew/pwn-env
```

- build from source

```sh
chmod +x ./build.sh && ./build.sh
```

Run install script

```sh
chmod +x ./install.sh && ./install.sh
```
